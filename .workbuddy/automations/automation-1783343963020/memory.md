# 自动化执行记忆 · AI自治多智能体项目

## 运行模式
- 方向驱动（Direction-Driven）模型（D20）。每次调用必须在 5 个开发方向（核心玩法／内容扩建／系统新创／体验打磨／工程基石）至少一个上推进。
- 方向1（核心玩法）冻结；方向2（内容扩建）最高优先级（≥一半调用优先做）；方向4 暂缓；方向5 全绿兜底。
- 禁止网络操作；纯 Node 零依赖冒烟/单测验证；根目录仅 5 控制/日志文件 + 仓库基础设施，游戏代码全在 `magic-arena/`。
