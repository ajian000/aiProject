## 3. 为什么游戏不到 5 分钟？— 严格计时分析

### 3.1 逐关时间拆解（easy 模式）

```
敌方血量缩放 0.60×，火球术伤害 25

第1关 "平原哨探"
  ├─ 看开场故事  → 15秒
  ├─ 战斗：敌方 45HP, 2 火球术杀一个, 3 个敌人 = 1 回合
  │    选人→移动→火球→下一人→移动→火球→下一人→火球  → 20秒
  ├─ 看胜利结算  → 10秒
  └─ 看结束故事  → 15秒
  小计: 60秒

第2关 "林间伏击"
  ├─ 战斗：同类敌人, 同上套路           → 20秒
  ├─ 故事 →结算→故事                   → 40秒
  小计: 60秒

第3-5关 "雪原/废墟/熔岩"
  ├─ 角色已升级(Lv2-4), 伤害+8~16
  ├─ 火球术 33-41 伤害, 杀得更快         → 15秒/关 × 3 = 45秒
  ├─ 抉择点(第3关+第5关各有一次二选一)   → 15秒
  小计: 60秒

第6关 "大魔导师决战"
  ├─ 角色 Lv5-6, 火球术 45 伤害
  ├─ Boss 150HP×0.60=90HP, 2 轮击杀     → 30秒
  ├─ 看终局故事 + 结局                   → 15秒
  小计: 45秒

总计: ~4分15秒
```

### 3.2 为什么战斗这么快

| 原因 | 数据 | 影响 |
|------|------|------|
| 每方仅 3 个单位 | 3v3 = 6 个单位在场上 | 每方一次行动只有 3 次操作 |
| 战场仅 8×8 | 移动范围 2-3 格 | 走位只有 2-3 步，几乎没有战术决策 |
| 玩家固定 3 技能/人 | 最优解明确 | 不需要思考，秒选最优方案 |
| 无兵种克制 | 所有人都是法师 | 不需要考虑对谁用什么策略 |
| 成长系统加速后期 | Lv6 时伤害翻倍 | 后期战斗比前期还快 |
| AI 瞬间决策 | 无动画等待 | 敌方回合几乎零耗时 |

### 3.3 内容消耗结构

```
玩家整个游戏体验 = 点击约 80 次鼠标

├─ 点击"战役"→"第1关"   = 2 次
├─ 点击"关闭"阅读剧情    = 1 次
├─ 战斗中点击操作(选人/移动/技能/目标)
│   × 6 关 × 约 15 次/关 = 90 次
├─ 点击"下一关"          = 5 次
├─ 点击抉择选项          = 2 次
└─ 看完结局              = 1 次

结论: 一个通关流程 ≈ 101 次鼠标点击、总耗时不到 5 分钟
```

## 10. 根本问题诊断

### 10.1 病根不在"加东西"，而在"规模错了"

```
链式反应：
                只有 3 个单位
                    ↓
            每方一次只有 3 次操作
                    ↓
          战斗 1-2 回合就结束了
                    ↓
          6 关 × 2 分钟 = 12 分钟
                    ↓
      玩家一小时内能通关 12 次
                    ↓
             游戏时长 < 5 分钟
```

### 10.2 项目投入方向有问题

开发历史（按代码行估算）：

```
方向1 核心玩法（技能/状态/AI/机制）   → ~800 行, 35%
方向3 系统新创（成就/图鉴/成长等）    → ~500 行, 22%
方向4 体验打磨（飘字/结算/日志/预览） → ~300 行, 13%
方向2 内容扩建（剧情/百科/支援对话）  → ~250 行, 11%
方向5 工程基石（测试/性能/建构建模）  → ~400 行, 17%
```

**60% 的代码花在"造引擎"上，只有 11% 花在"铺赛道"上。**
引擎造得再好，赛道 5 分钟跑完，玩家照样觉得这是个半成品。

## 11. 行动路线图

### 11.1 必须理解的核心矛盾

Magic Arena 要解决的根本问题不是"加什么功能"，而是：

> **战斗系统的承载能力（10 分）和内容的消耗速度（<5 min）之间存在数量级的错配。**

### 11.2 如何让游戏时长从 5 分钟 → 2 小时+

| 手段 | 当前 | 目标 | 影响 | 复杂度 |
|------|------|------|------|--------|
| **扩充章节** | 6 关 | 15-20 关 | 游戏时长 ×3 | 低 |
| **扩大战场** | 3v3 / 8×8 | 5v5 / 12×10 | 每关时长 ×2 | 中 |
| **增加战斗轮数** | 平均 2 轮 | 平均 5-6 轮 | 每关时长 ×3 | 低 |
| **插入兵种** | 纯法师 | 法+战+弓+奶 | 决策深度 ×2 | 高 |
| **插入装备系统** | 无 | 武器+饰品 | 战前准备维度 +1 | 中 |
| **插入外传** | 无 | 3-5 外传 | 游戏时长 ×1.5 | 中 |
| **插入营地系统** | 无 | 聊天/整备 | 玩法维度 +1 | 中 |

### 11.3 分阶段计划

```
Phase 0 — 止血（本阶段目标：让游戏超过 15 分钟）
├─ 章节：6 关 → 12 关（复用现有敌军池）
├─ 战斗规模：3v3 → 4v4
├─ 战场：8×8 → 10×10
├─ 平衡性：重新调整 difficulty HP/dmg 系数
└─ 预估效果：每关 3-5 分钟 × 12 关 ≈ 45-60 分钟

Phase 1 — 拉长（目标：1-2 小时）
├─ 兵种系统：战士(近战) / 弓手(远程) / 法师(范围) / 治疗
├─ 兵种克制：战克弓、弓克法、法克战（初版循环克制即可）
├─ 单位数：4v4 → 5v5
├─ 战场：10×10 → 12×10
├─ 新关卡类型：防守关 / 限回合关 / Boss 关 / 护送关
└─ 预估效果：每关 5-8 分钟 × 12 关 ≈ 1-1.5 小时

Phase 2 — 丰富（目标：2-3 小时）
├─ 装备系统（武器 + 饰品，商店购买）
├─ 转职系统（下位→上位，产生质变）
├─ 外传章节（3-5 条角色个人故事）
├─ 好感度系统（支援对话 → 属性加成）
└─ 预估效果：主线 1.5h + 外传 0.5h + 装备/转职 0.5h ≈ 2.5h

Phase 3 — 深挖（目标：5h+）
├─ 隐藏角色招募（特定条件解锁）
├─ 多周目要素（New Game+ / 隐藏难度）
├─ 关卡编辑/随机地图（无限可重玩性）
└─ 预估效果：主线 2h + 外传/隐藏 1h + 周目 2h ≈ 5h
```
